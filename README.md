labgrid-typings
===============

Type stubs for the [labgrid](https://github.com/labgrid-project/labgrid) library.

Usage
-----

Include this as a submodule in your project and point your type checker to it.

For example, Pylance/Pyright will look in `typings/` folders automatically.

    git submodule add https://github.com/dlech/labgrid-typings.git typings/labgrid
